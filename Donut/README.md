# 3D-spinning-ascii-donut-rendering donut
This is a donut that renders an 3D spinning ascii donut.

## Usage
```
> ./main
                                                            
                                                            
                                                            
                                                            
                                                            
                                                            
                     %%####******##%%                       
                  %###**++++++++++++**#%                    
                 #***+++++=+=======+=+***                   
               ***++++++==---------===+=+**                 
              **++++====--:::....::--====++*                
             *++++++===--::.    ...::--==+++                
             ++++======--:.      ..:---====++               
             +++=========-        ---======++               
             =========+++*       #**+++======               
             =======++++*##     @%##**+++====               
              -======+++**#%@@@@%%##*+++====                
              ---====++++**##%%%###**+++==--                
               :----==+++++********++===--                  
                 ::---===+=+===+++=+===-:                   
                   :::----=========--:.                     
                      ..::::::::::..                        
                                                            
                                                            
                                                            
                                                            
                                                            
                                                            
                                                            
```

for the best expirience use monospace font that has width to height ratio of 1:2 and
have the terminal window at least 60 characters wide and 30 characters high.
